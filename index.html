스네이크 게임

맵을 꽉 채우는 퍼펙트 클리어에 도전하세요!

게임 시작 🎮

📜 게임 방식

  • 꼬리를 늘려가며 점수를 올리는 클래식 게임입니다.
  • 화면을 100% 채우면 '진 엔딩'을 볼 수 있습니다!

⌨️ 조작법

  • PC: 방향키(상하좌우) 또는 W, A, S, D
  • 모바일: 화면 하단의 방향 버튼 터치
🧱

벽돌 깨기

스테이지 클리어부터 숨 막히는 무한 생존 모드까지!

게임 시작 🎮

📜 게임 방식

  • 스테이지 모드: 블록을 모두 깨고 다음 단계로 진입하세요.
  • 무한 모드: 블록이 계속 내려옵니다. 바닥에 닿기 전에 부수세요!
  • 노란색(+5) 아이템을 깨면 공이 5갈래로 퍼져나갑니다.

⌨️ 조작법

  • PC/모바일 공통: 패드를 마우스(또는 터치)로 누른 채 좌우로 드래그하여 이동
🦖

티렉스 런

점프와 급강하를 활용해 선인장과 익룡을 피하세요.

게임 시작 🎮

📜 게임 방식

  • 장애물을 무사히 통과할 때마다 1점씩 오릅니다.
  • 5점 이상부터는 하늘을 나는 '익룡'이 등장합니다.

⌨️ 조작법

  • 점프: 스페이스바, W, ↑, 좌측 화면 터치
  • 숙이기: Ctrl, S, ↓, 우측 화면 터치
  • 💡 팁 (급강하): 공중에 떴을 때 숙이기를 누르면 바닥으로 뚝 떨어지며 재빠르게 피할 수 있습니다!
💩

똥피하기

하늘에서 쏟아지는 똥과 파멸의 똥을 피해 살아남으세요!

게임 시작 🎮

📜 게임 방식

  • 떨어지는 똥을 피하며 오래 버틸수록 똥이 더 자주 내립니다.
  • 우산(1% 확률): 먹으면 6초간 무적이 되며 똥을 통통 튕겨냅니다.
  • 파멸의 일격: 2000점이 넘으면 캐릭터 머리 위로 빨간 경고선이 생긴 후, 엄청난 속도의 똥이 내리꽂힙니다!

⌨️ 조작법

  • PC: 좌우 방향키 (←, →) 또는 A, D
  • 모바일: 화면 하단의 좌측/우측 버튼 터치
