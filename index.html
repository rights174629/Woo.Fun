스네이크 게임

게임 시작 🎮

📜 게임 방식

  • 꼬리를 늘려가며 점수를 올리는 클래식 게임입니다.
  • 화면을 100% 채우면 '진 엔딩'을 볼 수 있습니다!

⌨️ 조작법

  • PC: 방향키(상하좌우) 또는 W, A, S, D
  • 모바일: 화면 하단의 방향 버튼 터치

벽돌 깨기

게임 시작 🎮

📜 게임 방식

  • 스테이지 모드: 블록을 모두 깨고 다음 단계로 진입하세요.
  • 무한 모드: 블록이 계속 내려옵니다. 바닥에 닿기 전에 부수세요!
  • 노란색(+5) 아이템을 깨면 공이 5갈래로 퍼져나갑니다.

⌨️ 조작법

  • PC/모바일 공통: 패드를 마우스(또는 터치)로 누른 채 좌우로 드래그하여 이동
🦖

티렉스 런

게임 시작 🎮

📜 게임 방식

  • 장애물을 무사히 통과할 때마다 1점씩 오릅니다.
  • 5점 이상부터는 하늘을 나는 '익룡'이 등장합니다.

⌨️ 조작법

  • 점프: 스페이스바, W, ↑, 좌측 화면 터치
  • 숙이기: Ctrl, S, ↓, 우측 화면 터치
  • 💡 팁 (급강하): 공중에 떴을 때 숙이기를 누르면 바닥으로 뚝 떨어집니다!
💩

똥피하기

게임 시작 🎮

📜 게임 방식

  • 떨어지는 똥을 피하며 오래 버틸수록 똥이 더 자주 내립니다.
  • 우산(1% 확률): 먹으면 6초간 무적이 되며 똥을 통통 튕겨냅니다.
  • 파멸의 일격: 2000점이 넘으면 캐릭터 머리 위로 엄청난 속도의 똥이 내리꽂힙니다!

⌨️ 조작법

  • PC: 좌우 방향키 (←, →) 또는 A, D
  • 모바일: 화면 하단의 좌측/우측 버튼 터치
🏎️

(공사중)

게임 시작 🎮

📜 게임 방식

  • 무한으로 펼쳐지는 부드러운 3D 평면 도로를 질주하세요.
  • 도로를 벗어나도 걱정 마세요, 즉시 복귀할 수 있습니다!

⌨️ 조작법

  • W / S : 가속 / 후진
  • A / D : 좌 / 우 핸들링
  • Space : 브레이크
  • R : 도로 중앙으로 즉시 복귀
  • P : 헤드라이트 ON/OFF
📈

그래프 계산기

계산기 실행 🎮

📜 핵심 기능

  • 좌측 수식 창과 우측 그래프 영역으로 나뉘어 있어 직관적으로 사용할 수 있습니다.
  • 수식을 추가하고 실시간으로 타이핑하여 여러 수식을 겹쳐 그릴 수 있습니다.
  • 곱하기(*)를 생략하고 2x, 3sin(x) 처럼 직관적으로 써도 자동 변환됩니다.

⌨️ 사용법

  • 수식 추가 : ➕ 버튼을 눌러 새 라인을 만들고 수식을 적으세요.
  • 범위 조절 : X 최소/최대 값을 변경하면 그래프가 실시간 변경됩니다.
  • 수식 삭제 : 칸 우측의 ✕ 버튼을 누르면 해당 그래프가 제거됩니다.
